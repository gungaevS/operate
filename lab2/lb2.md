# Лабораторная работа 2: Установка Linux

## Тема
Исследование процесса загрузки и установки операционной системы Linux методом ручной начальной загрузки.

## 1. Создание виртуальной машины
Создана ВМ в VirtualBox: x86_64, 2 ГБ ОЗУ, 20 ГБ HDD
Включена аппаратная виртуализация (AMD SVM)
Сетевой адаптер: NAT
Подключён ISO-образ System Rescue CD 64-bit

## 2. Загрузка Linux
Разметка диска: 
- /dev/sda1 (boot, 512M)
- /dev/sda2 (swap, 2G)
- /dev/sda3 (root, ~17.5G)

Базовая установка через debootstrap (Ubuntu 22.04 / jammy)
Настройка fstab, chroot, локали, часового пояса
Установка и настройка загрузчика GRUB

## 3. Настройка системы
Установлен и настроен SSH-сервер (sshd)
Создан пользователь с паролем
Настроен последовательный порт для удалённого управления через PuTTY
Проверено подключение

## 4. Настройка swap-пространства
Создан swap-файл размером 2 ГБ
Установлены права 600
Файл активирован командой swapon
Добавлена запись в /etc/fstab для автоматического подключения
Проверка: free -h
