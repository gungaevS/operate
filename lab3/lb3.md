# Лабораторная работа №1. Вариант 1

## Тема
Копирование всех изображений из указанного каталога и его подкаталогов в папку резервного хранения.

## Цель работы
Разработать bash-скрипт для автоматического резервного копирования изображений.

## Используемые команды
- `find` — поиск файлов
- `cp` — копирование файлов
- `mkdir` — создание каталога

## Краткое описание алгоритма
1. Принять параметры командной строки (источник и цель)
2. Если источник не указан — взять текущий каталог
3. Если цель не указана — создать папку `~/backup`
4. Найти все файлы с расширениями изображений (jpg, png, gif и др.)
5. Скопировать найденные файлы в целевую папку
6. Вывести результат работы

## Требования к окружению
- Операционная система: Linux
- Интерпретатор: bash

## Инструкция по запуску

# Дать права на выполнение
chmod +x backup_images.sh

# Запуск с указанием источника
./backup_images.sh
